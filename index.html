Utilizar el actual código fuente como base para realizar el siguiente ejercicio integrador de todas las tecnologías vistas hasta ahora: Utilización de Google (preferentemente en inglés), HTML5, CSS3, Bootstrap, Javascript, GIT, jQuery (incluido AJAX)

  1. Crear una base de datos en Firebase (Realtime Database).
  2. Crear un repositorio en github y vincularlo con su entorno local.
  3. Hacer funcionar el sitio en github pages.
  4. Crear un formulario que va a tener como objetivo consultarle a tus compañeros su opinión acerca de algo que te interese. Sería interesante utilizar algunos campos de formulario que no hayas utilizado hasta ahora y asignar ciertos valores numéricos a algunos campos para poder realizar algunos cálculos al recuperar la información luego
  5. Compartir con los compañeros la ubicación de su formulario en github pages así pueden rellenar el formulario en cuestión
  6. Crear una página de análisis de respuestas con los datos provistos por los compañeros
  7. Mejorar la estética de tu sitio con CSS, SASS y/o Bootstrap (lo que prefirais).
  8. Refactorizar el código lo más posible: modularizando lógica en diferentes archivos y encapsulando en funciones lo más posible (siempre siguiendo el principio DRY).
